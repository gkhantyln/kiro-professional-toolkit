## 🚀 NASIL KULLANILIR?

### 1. Manuel Hook Tetikleme
1. Sol panelde "Agent Hooks" bölümünü aç
2. İstediğin hook'u bul
3. ▶️ Play butonuna tıkla
4. Hook aktif olur, ben o uzman moduna geçerim

### 2. Otomatik Hook (Enterprise Dev Team)
- Spec task başladığında otomatik aktif olur
- Tam ekip yapısı ile çalışmaya başlarım

## 🎯 KULLANIM ÖRNEKLERİ

### Frontend Geliştirme
```
1. Mobile Dev Expert hook'unu aktif et
2. React Native component oluştur
3. Performance Expert ile optimize et
4. Accessibility Expert ile a11y kontrol et
```

### Backend API Geliştirme
```
1. API Design Expert hook'unu aktif et

4. Database Expert ile DB optimize et
```

### DevOps Pipeline
```
1. DevOps/SRE Expert hook'unu aktif et
2. CI/CD pipeline kur
3. Performance Expert ile monitoring ekle
4. Security Expert ile güvenlik kontrol et
```

### Blockchain Projesi
```
1. Blockchain/Web3 Expert hook'unu aktif et
2. Smart contract yaz
3. Security Expert ile audit yap
4. Tech Lawyer ile legal kontrol et
```

## 🔧 HOOK TİPLERİ

### userTriggered (Manuel)
- Sen tetiklediğinde çalışır
- Sol panelden play butonu ile aktif edilir
- İstediğin zaman açıp kapatabilirsin

### preTaskExecution (Otomatik)
- Spec task başlamadan önce otomatik çalışır
- Enterprise Dev Team bu tipte
- Otomatik ekip yapısı sağlar

## 💡 İPUÇLARI

### Hook Kombinasyonları
```
✅ İyi kombinasyonlar:

❌ Gereksiz kombinasyonlar:

- Birbiriyle çelişen uzmanlar
```

### Ne Zaman Hangi Hook?
- **Yeni proje**: Architecture Expert
- **API geliştirme**: API Design Expert
- **Mobil uygulama**: Mobile Dev Expert
- **Güvenlik odaklı**: Security Expert
- **Performance sorunları**: Performance Expert
- **Accessibility gerekli**: Accessibility Expert
- **Blockchain/DeFi**: Blockchain Expert
- **Ürün planlama**: Product Manager Expert

## 🔧 SORUN GİDERME

### Hook Görünmüyor
- `.kiro/hooks/` klasöründe olduğunu kontrol et
- Dosya uzantısının `.kiro.hook` olduğunu kontrol et
- Kiro'yu yeniden başlat

### Hook Çalışmıyor
- Play butonuna tıkladığından emin ol
- Hook JSON formatının doğru olduğunu kontrol et
- Kiro logs'unu kontrol et

### Hook Çok Yavaş
- Aynı anda çok fazla hook aktif etme
- Sadece ihtiyacın olan hook'ları kullan
