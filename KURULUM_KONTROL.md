```bash
install.bat

# Python
python install.py
```

### 4. Hooks Görünüyor mu?

```

```

- [ ] Yeni bir `.py` dosyası aç → Python + Django kuralları gelsin

## 🆘 SORUN GİDERME

### Hooks Görünmüyor
1. `.kiro/hooks/` klasörünü kontrol et
2. Dosya uzantılarının `.kiro.hook` olduğunu kontrol et
3. Kiro'yu yeniden başlat

### Skills Çalışmıyor
1. `.kiro/skills/` klasörünü kontrol et
2. `#skill-name` formatını kullan

### MCP Bağlanamıyor
1. `.kiro/settings/mcp.json` syntax'ını kontrol et
2. Command Palette → "MCP: Reconnect All Servers"
3. `disabled: false` olan server'ları kontrol et

### Steering Aktif Değil
1. `.kiro/steering/` klasörünü kontrol et
2. Dosya uzantısının `.md` olduğunu kontrol et
3. Front-matter `inclusion:` değerini kontrol et
